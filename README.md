# Frontend Monorepo

-   [NPM Workspaces](https://docs.npmjs.com/cli/v7/using-npm/workspaces)

## Installation

```sh
npm i '@rbkmoney/<PACKAGE_NAME>'
```

## Development / Usage

```sh
npm link '<PATH>/fe-core/packages/<PACKAGE_DIR>'
```

## Publish

1.  Bump package version
2.  ```sh
    npm publish
    ```
